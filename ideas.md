## Idea 1: "Precision Ledger" — Swiss Financial Modernism

**Design Movement:** Swiss International Style meets Bloomberg Terminal
**Core Principles:**
- Absolute information density with zero decorative noise
- Grid-based tabular layouts that echo the source spreadsheet
- Monospace numerics for scannable financial data
- High contrast, no gradients — every pixel earns its place

**Color Philosophy:** Near-black (#0F1117) background with crisp white text, a single accent of amber (#F59E0B) for key metrics and CTAs. Muted slate for secondary info. Evokes precision instruments and trading terminals.

**Layout Paradigm:** Left sidebar navigation with full-width content panels. No hero sections. Data tables and input grids dominate. Sidebar collapses to icons on mobile.

**Signature Elements:**
- Monospace font (JetBrains Mono) for all numbers
- Horizontal rule dividers with section labels in uppercase tracking
- Amber highlight bars on active rows/inputs

**Interaction Philosophy:** Instant feedback — every input change recalculates in real time. Hover states reveal secondary data. No modals; inline editing.

**Animation:** Minimal — only number count-up animations on projection results. No decorative motion.

**Typography System:** JetBrains Mono for numbers/data, IBM Plex Sans for labels and body. Tight line-height, generous letter-spacing on headers.

## Idea 2: "Horizon" — Warm Modernist Financial Planning

**Design Movement:** Mid-century modernism meets contemporary fintech (think Fidelity meets Notion)
**Core Principles:**
- Warmth and approachability without sacrificing clarity
- Generous whitespace with purposeful information hierarchy
- Sidebar-driven navigation with card-based content sections
- Subtle depth through layered cards and soft shadows

**Color Philosophy:** Warm off-white (#FAFAF8) background, deep forest green (#1B4332) as primary, amber (#D97706) as accent for positive projections, soft sage (#84A98C) for secondary elements. Evokes stability, growth, and nature — not cold finance.

**Layout Paradigm:** Persistent left sidebar (collapsible) with tabbed main content. Input sections use two-column form grids. Results use full-width chart + side-by-side summary cards.

**Signature Elements:**
- Rounded card containers with subtle drop shadows
- Green gradient progress bars for account growth
- Serif display font (Playfair Display) for section headings

**Interaction Philosophy:** Step-by-step guided flow (wizard-style) for first-time setup, then free-form editing. Tooltips explain financial concepts inline.

**Animation:** Smooth tab transitions, chart line draw-on animation, card entrance with subtle fade-up.

**Typography System:** Playfair Display for section titles, Inter for body/labels, tabular-nums for all financial figures.

## Idea 3: "Blueprint" — Technical Drafting Aesthetic

**Design Movement:** Engineering blueprint / technical documentation
**Core Principles:**
- Grid paper texture as background motif
- Structured, form-heavy layout with clear section delineation
- Data-forward: charts and tables are the heroes
- Utilitarian beauty — form follows function

**Color Philosophy:** Deep navy (#0A1628) background with blueprint blue (#2563EB) accents, white text, and cyan (#06B6D4) for chart highlights. Feels like a technical drawing come to life.

**Layout Paradigm:** Full-width single-page app with sticky top navigation. Sections stack vertically with clear visual breaks. No sidebar — horizontal tab navigation.

**Signature Elements:**
- Dotted grid background on input panels
- Dashed border cards with corner brackets
- Monospace labels with unit annotations

**Interaction Philosophy:** All inputs visible at once (no pagination). Recalculate button triggers animated chart updates. Keyboard-navigable.

**Animation:** Chart bars/lines animate in from left on calculate. Section transitions use horizontal slide.

**Typography System:** Space Grotesk for headings, Source Code Pro for numbers, system-ui for body text.

---

**Selected Design: Idea 2 — "Horizon" (Warm Modernist Financial Planning)**

Rationale: This approach balances the serious nature of retirement planning with an approachable, trustworthy aesthetic. The warm color palette and generous whitespace make complex financial data feel manageable rather than intimidating. The sidebar + tabbed layout maps naturally to the spreadsheet's two-sheet structure (Inputs + Budget), and the card-based design allows clean separation of input sections.
